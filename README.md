# 🌟 Flappy Bird Game

Fly high! 🚀  
A simple **Flappy Bird** clone with modern animations and interactive gameplay.

![Gameplay Preview](a.png)

## Features
- **Smooth Animations** 🌀 – Fluid bird movement & scrolling background
- **Responsive Controls** 🎮 – Tap/Spacebar to flap
- **Dynamic Starry Background** ✨ – Interactive falling stars
- **Lightweight & Fast** ⚡ – Pure HTML, CSS, JS

## How to Play
1. Open `index.html` in your browser.
2. Press **Spacebar** or **Click/Tap** to flap.
3. Avoid obstacles and score points!
